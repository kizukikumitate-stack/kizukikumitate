## 記事の共通構成

各記事は以下の流れで設計:

1. **キャッチーなリード（3〜5行）**: 読者の悩みを言語化、本書を読む理由を提示
2. **なぜ今この本か**: 現代の組織課題との接続
3. **本書の核となる主張**: 3〜4つのキーポイント
4. **私の現場視点**: ファシリテーターとしての実装経験
5. **対話力向上研修との繋がり**: 自社プログラムへの接続
