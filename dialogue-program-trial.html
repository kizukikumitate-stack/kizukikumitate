月例開催 / Try First

対話力向上研修を
1.5〜2時間で体感する
体験会・説明会

きづきくみたて工房が提供する対話力向上研修のエッセンスを、月例の体験会で実際にご体感いただけます。デモクラシーフィットネスの30分ワークを通じて、「対話の筋肉」を鍛えるとはどういうことか、ご自身の身体で体験してください。組織への導入を検討している人事・経営者の方も、ご自身の対話力を試したい個人の方も歓迎します。

Overview

体験会の概要

本格的な対話力向上研修の導入前に、まず1.5〜2時間で「対話力向上研修とは何か」を体感する場です。デモクラシーフィットネスの実ワークを2〜3種目体験し、その後にプログラム全体の解説と質疑応答を行います。

Schedule

当日のプログラム

2時間構成の場合の標準的な流れです。回によって内容を一部入れ替えることがあります。

0:00
オープニング・自己紹介
短いチェックインで場をひらきます。対話の前提となる「お互いの存在を認める」状態をつくります。
0:15
対話力向上研修とは何か（10分解説）
「対話」と「会話」の違い、なぜ筋トレ的アプローチが必要かを簡潔にお伝えします。
0:25
デモクラシーフィットネス 30分ワーク #1
「傾聴」「異見対応」「合意形成」など、10種目から1つを選んで実体験します。
1:00
振り返り・気づきの共有
体験を通じて気づいたこと、自組織で起きていることとの接続を対話します。
1:20
デモクラシーフィットネス 30分ワーク #2（任意）
時間に余裕がある回では2種目目を体験。種目ごとに鍛える筋肉が異なります。
1:50
クロージング・質疑応答
本格プログラムの導入を検討中の方からの質問にお答えします。組織状況に応じたカスタマイズの相談もこの時間で。

For Whom

こんな方に参加していただきたい

  • 人事・人材開発担当者
  • 経営者・役員・部門長
  • 組織開発・OD担当者
  • 研修発注の窓口担当者
対話力を高めたい個人の方
  • 1on1や会議のファシリテーションを担う管理職
  • チーム内の対話の質を上げたいリーダー
  • 心理的安全性に関心のある方
  • ファシリテーター・コーチとして活動している方
